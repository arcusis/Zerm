Section 04

About

Zerm is a native voice-to-clipboard tool for developers. Tap a key, speak, paste. Whisper transcribes on-device, Ollama can polish locally, and the workflow stays private by default.

Repository github.com/arcusis/Zerm Releases Downloads and changelog Issues Report bugs and requests Website Project homepage
Privacy

At runtime: everything happens on your machine. Your microphone audio, raw transcripts, and final outputs never leave the device. The only network call is to your local Ollama instance at 127.0.0.1:11434.

At first-run setup: Zerm downloads the Whisper model from huggingface.co (hash-pinned and verified) and launches the official Ollama release asset after hash verification and publisher checks where supported. You'll confirm before the Ollama installer runs.

Runtime

Auto-paste is opt-in. On macOS, Zerm restores the app that was focused when recording started, then sends a normal Cmd+V through Accessibility. Windows and Linux use the fixed Ctrl+Shift+Space hotkey while native paste support is expanded.

Open source

Zerm is MIT licensed and built in the open. Release builds run type checks and Rust tests on macOS, Windows, and Linux.
